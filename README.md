# Power BI Retail & Sales Analytics Dashboards

## üìä Overview

This Power BI project showcases five interactive dashboards developed during my Data Technician Bootcamp. Each dashboard demonstrates core analytics skills including:

- **Data transformation and cleaning** using Power Query  
- **DAX measures and calculated columns** for dynamic insights  
- **Interactive visuals** with slicers, filters, and drill-throughs  
- **Data storytelling** across retail and sales domains  
- **Visual diversity**: bar charts, line graphs, pie charts, maps, and more  

These dashboards were built using Excel and OData sources, with a focus on turning raw data into compelling business insights.
